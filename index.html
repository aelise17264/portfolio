This app is still being refined but the image link will take you to an article I wrote about our version of the app.

Authentication and Testing

This coding test challenged me to authenticate users and impletment unit tests. JSON webtokens and bycrypt middleware were used to authenticate users while protecting their information. For example, users' passwords were hashed upon entry via bycrypt and once hashed they could not be reverted to plain text. Jest was used to write unit tests to make sure that each step along the way was working correctly.
